Why LTX Video?

One of the fastest open text-to-video models. The distilled 2B version runs great on consumer GPUs (RTX 4090 / 3090 recommended, works with ~12–24 GB VRAM). Supports high FPS and good motion.

Quick local setup (easiest with Diffusers):

pip install diffusers torch accelerate imageio imageio-ffmpeg
python -c "
from diffusers import LTXPipeline
from diffusers.utils import export_to_video
import torch
pipe = LTXPipeline.from_pretrained('Lightricks/LTX-Video', torch_dtype=torch.bfloat16)
pipe.to('cuda')
# then generate...
"
